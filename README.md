# Dr. Mostaque Hassan — Personal Portal
The official portfolio and research portal of Dr. Mostaque Hassan. Features a dynamic showcase of SCIE/Scopus publications, academic courses taught, and technical skills bridging data science and cybersecurity.

This repository contains the source code for the personal academic and professional portfolio of **Dr. Mostaque Hassan**, an experienced AI researcher, Principal Data Scientist Instructor, and Founder of Algoguido Technologies. 

The portal serves as a centralized hub to showcase:
- **Research & Publications:** High-impact SCIE/Scopus-indexed journal papers (Frontiers in Plant Science, Scientific Reports, etc.) and patents.
- **Academic Credentials:** Details on his PhD, ongoing Post-Doctoral Fellowship, and 18+ years of teaching experience.
- **Technical Expertise:** A comprehensive breakdown of skills spanning Deep Learning, GANs, Network Security, and Cloud Computing.

### Tech Stack
- **Framework:** Next.js / React
- **Styling:** Tailwind CSS (Custom Dark/Light mode tokens)
- **Language:** TypeScript
